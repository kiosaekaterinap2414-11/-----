# Construction Site Management

Prototype of a desktop application for managing a construction site, created as an exam project.

## Project Description
The application is designed to manage construction works, teams, materials, and users.
It supports two roles: Administrator and Worker.

The prototype was created in Figma and includes interactive navigation between screens.

## Features
- User authentication (Administrator / Worker)
- Dashboard with list of works
- Search and filter works
- Add, edit, and delete works (Administrator)
- View statistics and reports
- Interactive navigation between screens
